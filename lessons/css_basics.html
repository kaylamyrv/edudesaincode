CSS Dasar: Styling Halaman Web

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan HTML. Pelajari properti dasar untuk membuat halaman yang menarik dan responsif.

Properti Utama

  • color: Warna teks.
  • background-color: Warna latar belakang.
  • font-size: Ukuran font.
  • margin/padding: Spasi luar/dalam.

Contoh Kode

p {
  color: #ffffff;
  background-color: #6c5ce7;
  font-size: 16px;
  padding: 10px;
}
