HTML Dasar: Struktur Halaman Web

HTML (HyperText Markup Language) adalah fondasi web. Pelajari tag dasar untuk membuat struktur halaman yang solid dan semantik.

Tag Utama

  • <html>: Root elemen.
  • <head>: Metadata dan link eksternal.
  • <body>: Konten utama yang terlihat.
  • <h1> - <h6>: Heading untuk hierarki.
  • <p>: Paragraf teks.

Contoh Kode

<!DOCTYPE html>
<html>
<head>
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
  <p>Ini paragraf pertama.</p>
</body>
</html>
