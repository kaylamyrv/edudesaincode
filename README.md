# EduDesignCode

Website pembelajaran interaktif untuk **Pemrograman Dasar** (HTML, CSS, JS) dan **UI/UX Design**. Dibuat dengan HTML, CSS, JS murni, dan di-host gratis di GitHub Pages.

## Fitur

- Modul materi ringkas dengan contoh kode.
- Playground HTML/CSS untuk praktik langsung.
- Kuis otomatis per topik.
- Tema nuansa galaxy untuk pengalaman visual yang menarik.
- Responsive design.

## Cara Setup & Run

1. Clone repo: `git clone https://github.com/username/edudesigncode.git`
2. Buka `index.html` di browser.
3. Untuk deploy: Push ke GitHub dan aktifkan Pages di Settings > Pages (branch main, folder root).

## Struktur Folder

- `index.html`: Halaman utama.
- `about.html` & `contact.html`: Halaman statis.
- `lessons/`: Modul materi.
- `assets/`: CSS, JS, gambar.
- `quiz/`: Data kuis JSON.
